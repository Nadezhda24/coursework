***Перевод из одной системы счисления в другую***
====================================
**Цель** : реализовать программу, которая переводит из одной системы счисления в другую.
-----------
**Задача**:  
 Pеализовать калькулятор программист, поддерживающий :
 - Системы счисления :
	* двоичная
	* восьмеричная
	* десятичная
	* шестнадцатеричная.
