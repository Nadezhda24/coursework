 Введенное выражение: 2 / 3 

 1.Преобразование выражения из инфиксной записи в постфиксную
 для определения последовательности действий : 
 2 3 / 
 2.Подсчет выражения : 
 Перевод числа 3 в 10-ую систему счисления : 
3 * 8^0= 3
 Перевод числа 2 в 10-ую систему счисления : 
2 * 8^0= 2
 Перевод числа 3 в 8-ую систему счисления : 
 Перевод целой части числа: 
3 % 8 = 3
 ( 3 - 3 )  / 8 = 0
 Перевод числа 2 в 8-ую систему счисления : 
 Перевод целой части числа: 
2 % 8 = 2
 ( 2 - 2 )  / 8 = 0
