 для определения последовательности действий : 
 55 3 -

3 * 8^0= 3
 Перевод числа 55 в 10-ую систему счисления : 
5 * 8^0 + 5 * 8^1= 45
 Перевод числа 3 в 8-ую систему счисления : 
 Перевод целой части числа: 
3 % 8 = 3
 ( 3 - 3 )  / 8 = 0
 Перевод числа 45 в 8-ую систему счисления : 
 Перевод целой части числа:

 ( 45 - 5 )  / 8 = 5
